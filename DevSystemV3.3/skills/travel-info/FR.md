# France (FR)

French flights, trains, and public transport.

## When to Use This File

- Query mentions: France, FR, Paris, Lyon, Marseille, SNCF, Air France, CDG, ORY
- Query is about: French flights, trains, or transit

## Flights

### Paris CDG Airport
- **URL:** https://www.parisaeroport.fr/en/passengers/flights/flight-information
- **Best for:** CDG departures/arrivals

### Paris Orly Airport
- **URL:** https://www.parisaeroport.fr/en/passengers/flights/flight-information
- **Best for:** ORY departures/arrivals

### Flightradar24 (France filter)
- **URL:** https://www.flightradar24.com
- **Best for:** Live flights over France

## Trains

### SNCF Connect
- **URL:** https://www.sncf-connect.com/en-en/train-ticket/timetables
- **Note:** Blocks automated access - use browser or app
- **Best for:** TGV, regional trains

### SNCF App
- **Best for:** Real-time TGV status (web blocked, use app)

### Eurostar (France connections)
- **URL:** https://www.eurostar.com/us-en/travel-info/service-information
- **Best for:** Paris-London, Paris-Brussels

### Track-Trace France
- **URL:** https://track-trace.live/trainradar/trains-france/
- **Best for:** Visual train tracking

## Public Transport

### Paris (RATP)
- **URL:** https://www.ratp.fr
- **Best for:** Metro, RER, bus, tram
- **Example:** "Paris metro line 1 status"

### Lyon (TCL)
- **URL:** https://www.tcl.fr
- **Best for:** Lyon metro and tram

### Marseille (RTM)
- **URL:** https://www.rtm.fr
- **Best for:** Marseille public transport
