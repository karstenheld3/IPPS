# Trains (European Train Tracking)

Train tracking, delays, and station boards.

## When to Use This File

- Query mentions: train, rail, railway, station
- Query is about: Train status, delays, live tracking
- No specific country mentioned

## Live Train Maps

### Zugfinder Europe Map
- **URL:** https://www.zugfinder.net/en/livemap-europa
- **Coverage:** DE, AT, CH, BeNeLux, IT, DK, SI
- **Best for:** Real-time train positions, delay statistics
- **Example:** "Where is ICE 123?"

### Zugfinder Station Board
- **URL:** https://www.zugfinder.net/en/stationboard
- **Best for:** Departures and arrivals at any station

### Track-Trace Trainradar
- **URL:** https://track-trace.live/trainradar/
- **Coverage:** DE, FR, UK, AT, CH, IT, RO
- **Best for:** Visual multi-country train tracking
