# United Kingdom (UK)

British flights, trains, and public transport.

## When to Use This File

- Query mentions: UK, Britain, England, London, Manchester, Edinburgh, Heathrow, Gatwick
- Query is about: British flights, trains, or transit

## Flights

### Heathrow Airport
- **URL:** https://www.heathrow.com/departures
- **Best for:** LHR departures/arrivals

### Gatwick Airport
- **URL:** https://www.gatwickairport.com/flights/
- **Best for:** LGW departures/arrivals

### Manchester Airport
- **URL:** https://www.manchesterairport.co.uk/flight-information/
- **Best for:** MAN departures/arrivals

### Flightradar24 (UK filter)
- **URL:** https://www.flightradar24.com
- **Best for:** Live flights over UK

### Eurostar (UK connections)
- **URL:** https://www.eurostar.com/us-en/travel-info/service-information

### Trainline UK
- **URL:** https://www.thetrainline.com

### Track-Trace UK

- **URL:** https://developer.transportapi.com
