# Germany (DE)

German flights, trains, and public transport.

## When to Use This File

- Query mentions: Germany, DE, Berlin, Munich, Frankfurt, Hamburg, Cologne, DB, Lufthansa
- Query is about: German flights, trains, or transit

## Flights

### Frankfurt Airport
- **URL:** https://www.frankfurt-airport.com/en/flights-and-transfer/departures.html
- **Best for:** FRA departures/arrivals, gate info

### Munich Airport
- **URL:** https://www.munich-airport.de/en/consumer/flug-info/
- **Best for:** MUC departures/arrivals

### Flightradar24 (Germany filter)
- **URL:** https://www.flightradar24.com
- **Best for:** Live flights over Germany

## Trains

### Deutsche Bahn (DB)
- **URL:** https://www.bahn.de
- **Best for:** Journey planning, ticket booking
- **Example:** "Berlin to Munich train"

### DB Real-time Alerts
- **URL:** https://www.bahn.de/service/fahrplaene/aktuelle_meldungen
- **Best for:** Current disruptions, platform changes

### Zugfinder Germany
- **URL:** https://www.zugfinder.net/en/start
- **Best for:** Live train positions, delay statistics

### Track-Trace Germany
- **URL:** https://track-trace.live/trainradar/trains-germany/
- **Best for:** Visual train tracking

## Public Transport

### Berlin (BVG)
- **URL:** https://www.bvg.de
- **Best for:** U-Bahn, S-Bahn, bus, tram

### Munich (MVV)
- **URL:** https://www.mvv-muenchen.de
- **Best for:** Munich metro and regional transport

### Hamburg (HVV)
- **URL:** https://www.hvv.de
- **Best for:** Hamburg public transport

### Frankfurt (RMV)
- **URL:** https://www.rmv.de
- **Best for:** Frankfurt and Rhein-Main region

## Free APIs

### Deutsche Bahn Open Data
- **URL:** https://developers.deutschebahn.com
- **Cost:** Free with registration
- **Data:** Timetables, stations, real-time updates
