# Transit (City Public Transport)

Buses, metros, trams, and local transport.

## When to Use This File

- Query mentions: bus, metro, subway, tram, transit, public transport
- Query is about: City transport, local connections
- No specific country mentioned

## Multi-City Apps

### Moovit
- **URL:** https://moovitapp.com
- **Coverage:** 112+ countries, most European cities
- **Best for:** Real-time bus/metro/tram arrivals
- **Example:** "Next bus to central station"

### Citymapper
- **URL:** https://citymapper.com
- **Coverage:** Major cities (London, Paris, Berlin, etc.)
- **Best for:** Clean interface, excellent service alerts

### Transit App
- **URL:** https://transitapp.com
- **Coverage:** 200+ cities
- **Best for:** Live tracking, clean interface

### Google Maps Transit
- **URL:** https://www.google.com/maps
- **Coverage:** Global
- **Best for:** Universal fallback when other apps fail

## Free Transit APIs

### GTFS / GTFS Realtime
- **Spec:** https://gtfs.org
- **Feed database:** https://transitfeeds.com
- **Cost:** Free, open data
- **Data:** Routes, stops, schedules, real-time positions

### Navitia
- **URL:** https://www.navitia.io
- **Cost:** 90,000 requests/month free
- **Data:** Journey planning, French transit

### Transport API (UK)
- **URL:** https://developer.transportapi.com
- **Cost:** 1000 requests/day free
- **Data:** UK trains, buses, live departures
