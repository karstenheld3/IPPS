# Europe (Pan-European Resources)

Cross-border and pan-European travel resources.

## When to Use This File

- Query mentions: Eurostar, Thalys, cross-border, Europe-wide
- Query is about: International trains, pan-European flights

## Cross-Border Trains

### Eurostar
- **URL:** https://www.eurostar.com/us-en/travel-info/service-information
- **Routes:** London - Paris - Brussels - Amsterdam - Cologne
- **Best for:** Channel Tunnel trains, live departures
- **Example:** "Eurostar London to Paris status"

### Thalys / NS International
- **URL:** https://www.nsinternational.com/en
- **Routes:** Paris - Brussels - Amsterdam - Cologne
- **Best for:** Benelux high-speed connections

### Rail Europe
- **URL:** https://www.raileurope.com
- **Best for:** Eurail passes, cross-border planning

### Trainline
- **URL:** https://www.thetrainline.com
- **Best for:** Booking UK and European trains

## Pan-European Train Tracking

### Zugfinder Live Map
- **URL:** https://www.zugfinder.net/en/livemap-europa
- **Coverage:** Germany, Austria, Switzerland, BeNeLux, Italy
- **Best for:** Real-time train positions across Central Europe

### Track-Trace Trainradar
- **URL:** https://track-trace.live/trainradar/
- **Coverage:** DE, FR, UK, AT, CH, IT
- **Best for:** Visual multi-country train tracking

## Pan-European Flight Tracking

### FlightStats Airport Delays
- **URL:** https://www.flightstats.com/v2/airport-delays
- **Best for:** Europe-wide airport delay overview

### Flightradar24
- **URL:** https://www.flightradar24.com
- **Best for:** Live aircraft positions across Europe

## Multi-Modal Journey Planners

### Omio
- **URL:** https://www.omio.com
- **Best for:** Comparing trains, buses, flights across Europe

### Rome2Rio
- **URL:** https://www.rome2rio.com
