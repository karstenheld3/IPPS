## Steps

1. Re-read entire conversation, extract:
   - New problems → PROBLEMS.md
   - Important findings → NOTES.md
   - What worked/didn't → PROGRESS.md
2. Use `[TESTED]` and `[ASSUMED]` markers - never document unverified assumptions
3. Compare changes since last commit, summarize → PROGRESS.md
4. Update all session files
5. Run `/commit`
