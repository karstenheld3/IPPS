# Write STRUT Workflow

Create or insert a STRUT plan into any document.

## Required Skills

- @write-documents for STRUT template and formatting rules

## MUST-NOT-FORGET

- Run `/verify` to validate STRUT structure

## Steps

1. Read `STRUT_TEMPLATE.md` from @write-documents skill folder
2. Determine location: `PROGRESS.md`, `_IMPL_*.md`, `_TASKS_*.md`, or `NOTES.md`
3. Create STRUT following `STRUT_TEMPLATE.md` structure exactly
4. Run `/verify` (Planning context)
