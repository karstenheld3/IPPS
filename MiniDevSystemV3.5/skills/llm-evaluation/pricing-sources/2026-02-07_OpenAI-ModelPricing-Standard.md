## Pricing Notes

- Reasoning tokens are not visible via the API but occupy context window space and are billed as output tokens.
- For gpt-image-1.5, text output tokens include model reasoning tokens.
- Priority tier = faster processing. Flex tier = lower prices, higher latency. Batch API = additional savings for non-time-sensitive requests.
